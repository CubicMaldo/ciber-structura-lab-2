# Iconos de Logros - Placeholders

Este directorio debe contener los iconos para cada categoría de logro.

## Archivos necesarios:
- achievement_mission.png (64x64px recomendado)
- achievement_precision.png
- achievement_efficiency.png
- achievement_progression.png
- achievement_stage.png
- achievement_secret.png
- achievement_default.png

## Recomendaciones de diseño:
- Tamaño: 64x64 píxeles (o 128x128 para alta resolución)
- Formato: PNG con transparencia
- Estilo: Consistente con el tema del juego
- Colores: Coincidir con los colores de categoría:
  * Misiones: Azul
  * Precisión: Dorado
  * Eficiencia: Verde
  * Progresión: Púrpura
  * Fases: Naranja
  * Secretos: Rosa
  * Default: Gris

## Herramientas sugeridas:
- GIMP (gratis)
- Inkscape (vectorial, gratis)
- Aseprite (pixel art)
- Recursos gratuitos: flaticon.com, game-icons.net
